Hypervisor

Ein Hypervisor, auch als Virtual Machine Monitor (VMM) bekannt, ist eine Software, die es ermöglicht, mehrere virtuelle Maschinen (VMs) auf einer einzigen physischen Hardware auszuführen. Jede dieser VMs kann ein eigenes Betriebssystem und eigene Anwendungen betreiben, unabhängig von den anderen.

  • Typ 1: Bare-Metal Hypervisor
  • Typ 2: Hosted Hypervisor
Merksatz: Ein Hypervisor verwaltet Ressourcen wie CPU, Arbeitsspeicher und Speicherplatz für virtuelle Maschinen (VMs) und sorgt für deren Isolation.

NIC (Network Interface Controller)

NIC steht für Network Interface Controller und stellt die Verbindung zwischen virtuellen Maschinen und dem Netzwerk her.
